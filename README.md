# Kebab: The Ultimate Health Food

A static demo page making the science-backed case for kebab as a health food.

Live at: **https://rubendw.github.io/kebab-health/**

## Structure

```
index.html   — markup
style.css    — all styles and animations
script.js    — floating background kebab generator
```

## Running locally

```bash
python3 -m http.server 4000
```

Then open http://localhost:4000.

## Deploying

Pushes to `master` are automatically served by GitHub Pages.
